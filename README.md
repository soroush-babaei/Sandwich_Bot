# 🍔 Online Food Ordering Bot | Python & Telegram

یک ربات هوشمند سفارش‌گیری غذا که فرآیند سفارش‌دهی را در تلگرام کاملاً خودکار و آسان می‌کند. 🚀

## 🎯 درباره پروژه
این ربات با استفاده از پایتون و API تلگرام طراحی شده تا تجربه سفارش غذا را برای مشتریان و مدیریت رستوران را بهینه کند. از انتخاب منو تا صدور فاکتور و ارسال به ادمین، همه چیز در یک محیط یکپارچه انجام می‌شود.

## ✨ قابلیت‌های کلیدی
- 📋 **منوی دسته‌بندی‌شده:** دسترسی آسان به دسته‌های غذا، نوشیدنی و پیش‌غذا.
- 🛒 **سبد خرید هوشمند:** افزودن، حذف، ویرایش آیتم‌ها و تغییر تعداد در لحظه.
- 🔄 **تعویض محصول:** امکان تغییر سریع آیتم‌های انتخاب شده.
- 💰 **محاسبه خودکار:** محاسبه دقیق مبلغ نهایی سفارش.
- 📍 **ثبت آدرس:** امکان وارد کردن آدرس به صورت دستی یا ارسال موقعیت GPS.
- 📞 **اطلاعات تماس:** دریافت شماره تماس کاربر برای هماهنگی.
- 🧾 **فاکتور گرافیکی:** تولید و ارسال فاکتور تصویری زیبا برای مشتری.
- 📤 **ارسال به ادمین:** ارسال کامل جزئیات سفارش به مدیر رستوران.
- 👥 **پشتیبانی چندکاربره:** مدیریت همزمان چندین سفارش با پایداری بالا.

## 🛠️ ساختار پروژه
- `main.py`: هسته اصلی اجرای ربات.
- `.env`: فایل تنظیمات محرمانه (شامل توکن و آیدی ادمین).
- `.gitignore`: فایل‌های نادیده گرفته شده توسط گیت.
- `fonts/`: پوشش مربوط به فونت‌های گرافیکی (برای فاکتورها).
- `requirements.txt`: لیست کتابخانه‌های مورد نیاز.

## ⚙️ نحوه نصب و اجرا

1. **کلون کردن رپازیتوری:**
   ```bash
   git clone https://github.com/soroush-babaei/Sandwich_Bot.git
   cd Sandwich_Bot
   ```
2. **نصب وابستگی‌ها:**
   ```bash
   pip install -r requirements.txt
   ```
3. **تنظیمات:**
   ```bash
   TOKEN='Put Your Token'
   ADMIN_CHAT_ID='Put Your Admin Chat ID'
   ```
4. **اجرای ربات:**
   ```bash
   python main.py
   ```
